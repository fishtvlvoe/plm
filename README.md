# PLM (Personal Lab Manager) Skill

This repository contains the `plm` skill configuration, references, and knowledge bases designed for the Personalization Lifecycle Modeling process.

## Structure

- `SKILL.md`: The core skill configuration containing the phase-based extraction and update workflow.
- `references/`:
  - `extraction-framework.md`: Rules for personality trait extraction, confidence tagging (⚙️ / 🧪), and validation.
  - `output-templates.md`: Templates for generating `autobiography.md` and `digital-twin-guide.md`.
- `knowledge/`:
  - `data-sources.md`: Defines source file paths, formats, categories, and mapping rules.

## Runtime Compatibility

- Claude Code / Claude Desktop Skills
- Codex AGENTS.md-compatible skill runners
